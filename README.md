DevSecOps CI/CD Pipeline with AKS Deployment

Project Overview

This project demonstrates a complete DevSecOps CI/CD pipeline built using Azure cloud services and Kubernetes.
The pipeline automates:

* Code Quality Analysis
* Vulnerability Scanning
* Docker Image Build & Push
* Kubernetes Deployment
* Monitoring and Observability

The application is containerized using Docker and deployed to Azure Kubernetes Service (AKS) through Azure DevOps Pipelines.
